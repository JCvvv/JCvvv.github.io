前言

很难想象一个 985 院校组织的集体实习，只有听讲座和去企业走马观花，而且还要为此写一篇实验报告以换取学分。虽然没参观什么有名气的大厂，只在学校周边这些小厂逛一圈，但看他们绞尽脑汁希望给 985 学生留下好印象的努力，像极了企图填满简历的我，莫名的戳中了我的笑点。

课程安排的可以说是一坨，想用讲座来扩展学生的视野，真不知道是哪个天才想出来的，更别说学校都凑不出一个能容纳整个年级的报告厅，把高贵的资本家请到学校里来挑选未来的廉价劳动力。但是为了学分，我勉为其难的将这些讲座做了整理，方便日后杜撰我的实习报告。

全部的ppt都有记录，但如果很不幸你是21级并且在交作业之前看到这篇帖子，那么我存放这些资料的仓库还没有对外公开，但是我会简单的记录每场讲座大概讲了什么，帮助你恢复一点记忆完成作业。

讲座

信创

  1. 信创是信息技术应用创新产业,包括IT基础设施、基础软件、应用软件和信息安全等领域。自1993年以来,经过预研开端、国产实践和试点阶段,现已进入应用落地阶段。

  2. 信创产业呈现出中国电子、电科、华为等“三大四小”的发展格局。产业链以CPU和操作系统为核心,还包括数据库、中间件等环节。目前国产CPU以龙芯、飞腾、海光为主,操作系统以麒麟、统信为主。

  3. 信创对国家安全至关重要。2022年出台文件要求央企国企到2027年完成信创替代。信创也面临生态薄弱、产品差距等挑战。

  4. 信创未来保持高速增长,预计2026年规模将突破2000亿。“两弹一星”精神预示着我国信息产业必将成功走向自主可控。

CAD

  1. 工业软件对国家制造业至关重要,CAD是其核心。CAD技术从初期的线框模型,发展到曲面模型和实体模型,再到参数化模型和直接建模。

  2. 目前CAD核心技术有三维建模引擎、几何约束求解器、参数化机制、大规模装配渲染等。主要CAD系统有CATIA、NX、Creo、SolidWorks等。

  3. CAD未来发展趋势是多人协同、人机协同、集成化和智能化。软件也向云架构发展。

  4. 华天软件是国内CAD领域的先驱,自主研发了三维CAD系统Sinovation。最近又研发出国内首款云架构CAD系统CrownCAD,实现了核心技术的自主可控。

  5. CrownCAD通过自主可控的核心技术,实现了云架构下的协同设计、大规模装配、工程制图等功能。已经在多家企业和院校取得应用。

  6. 华天软件希望通过完整的工业软件产品线,为国产制造业提供强有力的智能化支持,推动中国制造向智造转型。

商用密码

  1. 商用密码是用于保护不属于国家秘密的商业信息的密码技术。它不同于口令,可以实现信息的加密保护和安全认证。

  2. 商用密码从古典密码发展到现代密码,现有各类对称密码、非对称密码、哈希密码等算法。它广泛应用于电子政务、金融交易、交通、票务等领域。

  3. 近年来,我国非常重视网络安全和数据安全,出台多项法律要求使用商用密码技术保护关键信息基础设施。

  4. 商用密码的未来发展趋势是与新技术深度融合,构建全面的网络安全防护体系。

  5. 三未信安是国内商用密码领域的龙头企业,拥有领先的技术和完善的产品线,包括密码卡、服务器密码机、密钥管理等。已经在政府、金融、电信等领域取得客户应用。

  6. 三未信安积极响应国家网络安全和数据安全战略,为客户提供安全可靠的商用密码技术和产品,保障信息基础设施安全。

知识图谱

  1. 知识图谱是一种用于知识表达和管理的语义网络,可以有效组织概念之间的关系。

  2. 知识图谱可以应用于医疗、金融、法律等多个领域,帮助知识管理和问题解决。演讲者重点介绍了医疗领域知识图谱的构建。

  3. 构建知识图谱的关键技术包括知识表示、知识获取、知识融合、知识推理等。需要深度学习、信息抽取等技术的支持。

  4. 演讲者通过医疗领域知识图谱的案例,介绍了本体建模、知识抽取、查询问答等知识图谱的具体应用。

  5. 知识图谱技术还有很多挑战,需要持续完善知识表示能力、推理能力、可解释性等方面。

  6. 知识图谱技术与人工智能结合,可以实现更好的语义理解和知识驱动,是未来智能系统发展的重要方向。

浪潮

  1. 低代码开发平台可以通过可视化操作实现业务应用的快速开发,大幅降低企业数字化的门槛。

  2. 低代码平台关键特性包括组件库、多屏支持、动态建模等。涵盖从可视化设计到代码生成的整个应用开发周期。

  3. 大数据平台通过采集、存储、计算、分析和可视化等功能,实现对海量数据的管理应用。核心技术有NoSQL、流计算等。

  4. 浪潮公司搭建了低代码平台和大数据平台,可实现业务应用、数据应用的快速开发。已经获得多地政府和企业的应用。

  5. 低代码平台可快速实施数字政务、智慧城市等应用。大数据平台可支持数据治理、可视化分析、智能决策等。

  6. 低代码与大数据平台的结合,可进一步提升数据价值,实现业务与数据的深度融合。这是未来政企数字化建设的重要趋势。

普联软件股份有限公司

主营： 应用软件 、平台软件 、集团管控 、数据应用 、数字建造

简介：普联软件股份有限公司（股票代码：300996），是大型集团企业管理信息化综合方案提供商，国家认定的高新技术企业、国家鼓励的重点软件企业。公司创立于2001年，在北京、深圳、成都、西安、武汉、合肥、乌鲁木齐、库尔勒、香港、日本大阪等地设有区域机构和服务团队。公司主要面向大型集团企业提供自主研发的软件产品和解决方案。公司提供的软件产品既有应用软件，也有平台软件。同时，面向大型集团企业的个性化需求和信息系统的集成需求，提供定制化解决方案，主要聚焦于集团管控、XBRL数据应用、平台软件等领域。正在积极拓展智慧工地、数字建造等建筑行业产业互联网相关业务。普联软件以研究企业、服务企业为己任，坚持聚焦大型集团客户、聚焦优势业务领域的发展策略。已形成以石油石化、建筑地产两大优势行业为核心，多业务领域客户群体不断扩大的新格局。公司拥有自主技术平台，支撑ToB模式下的产品研发、大规模客户化定制交付。公司自主构建的面向大型复杂架构系统部署需求的PaaS云平台、面向大型定制软件项目交付的OSP云开发平台，采用云原生技术，支持企业混合云架构，通过资源共享、共用和有效积累，提升交付能力、交付效率和交付质量，实现了以同一个技术底座、同一个开发平台满足集团企业构建数字化转型云平台的需要。始终紧跟技术发展趋势，不断提升研发创新能力，形成了紧贴客户需求的研发机制。
